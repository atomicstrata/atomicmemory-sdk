# AtomicMemory SDK has moved

This repository is deprecated. The canonical public source for the AtomicMemory TypeScript SDK now lives in the AtomicMemory monorepo:

- Source: <https://github.com/atomicstrata/atomicmemory/tree/main/packages/sdk>
- Issues: <https://github.com/atomicstrata/atomicmemory/issues>
- Docs: <https://docs.atomicstrata.ai/sdk>
- npm: <https://www.npmjs.com/package/@atomicmemory/sdk>

The package name remains `@atomicmemory/sdk`, and existing published versions remain available on npm. New source changes, issues, and release work should happen in `atomicstrata/atomicmemory`.

This split repository is retained temporarily for historical references during the public monorepo cutover.
